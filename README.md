## Taller LINQ Completo – C#
Este repositorio contiene las tres partes del taller práctico sobre LINQ:
LINQ to Objects, LINQ con clases personalizadas y LINQ to SQL.
Cada parte fue desarrollada en C# utilizando Visual Studio.

## Contenido del Repositorio

El proyecto está dividido en tres ejercicios principales:

** Parte 1 – LINQ to Objects

En esta sección practico consultas básicas usando listas simples.
Incluye ejemplos como:

Filtrar números

Obtener múltiplos

Ordenar valores

Mostrar resultados en consola

** Parte 2 – LINQ con Clases Personalizadas

Aquí trabajo con una clase Estudiante, aplicando LINQ sobre una lista de objetos.
Consultas implementadas:

Estudiantes menores de 21

Estudiantes con ID múltiplo de 2

Ordenados por edad

Nombre más corto y nombre más largo

** Parte 3 – LINQ to SQL

En esta parte conecto Visual Studio con SQL Server para consultar datos desde una base de datos real (EscuelaDB).
Consultas realizadas:

Estudiantes menores de 20

Estudiantes ordenados por nombre

Nombres que comienzan con la letra A

Incluye el archivo DataClasses.dbml, generado automáticamente por Visual Studio.

## Tecnologías Utilizadas

C#

.NET Framework 4.7.2

SQL Server Management Studio (SSMS)

LINQ to SQL

Visual Studio 2022
